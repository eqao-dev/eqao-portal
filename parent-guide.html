保護者の方へ

EQAO PORTALのご案内

family_restroom ポータルをご利用いただく保護者の皆様へ

EQAO PORTALは、生徒だけでなく保護者の皆様にもご利用いただけるよう設計されています。

保護者の方は、お子様と同じログインID・パスワードでログインいただけます。
新規でアカウントを作成する必要はありません。

このページでは、保護者の皆様がポータルをどのようにご活用いただけるか、また、ご家庭での関わり方についてご説明します。

visibility
ポータルで確認できる内容

ポータルにログインいただくと、以下のような情報をご確認いただけます。

school 併願校・出願校の状況および進捗
description 書類作成・課題・問題集の進捗状況
auto_stories 授業外学習の量・内容
assignment 宿題の内容
groups 担任・スタッフ間の会議で共有されている指導上の報告内容
event 各種面談（保護者面談・三者面談・塾長面談）の案内
video_camera_front 集団授業・イベント等へのアクセス情報
info

これらの情報は、お子様の学習状況を把握し、必要なときにサポートできる体制を整えるためにご活用ください。

handshake
ご家庭での関わり方について

総合型選抜の準備においては、お子様自身が主体的に取り組むことが重要です。そのため、保護者の皆様には以下のようなスタンスでのご協力をお願いしております。

ポータルの目的は「介入」ではなく「現状把握と連携」です。

check_circle 状況を把握し、必要なときにサポートできる状態を保つ
check_circle お子様の自主性を尊重しながら、見守る姿勢を大切にする
check_circle 塾との連携を通じて、適切なタイミングで関わる
warning

細かく介入しすぎることは、お子様の主体性を損なう可能性があります。一方で、何も把握していない状態も望ましくありません。適度な距離感で見守っていただくことをお勧めします。

folder_special
保護者向け情報について

ポータル内では、生徒が日常的に確認すべき情報と、保護者向けの情報を分けて表示しています。

person 生徒向け：日々の学習に必要な教材・スケジュール・連絡事項
family_restroom 保護者向け：ルール・規約の確認、各種チェックリスト、面談案内など
lightbulb

保護者の皆様には、下記のリンクから関連情報をご確認いただけます。

favorite ご協力のお願い

総合型選抜は、お子様自身の経験・考え・表現力が問われる入試です。

保護者の皆様には「見守り・連携」を前提としたポータル利用をお願いしております。お子様が自分の力で成長できるよう、適度な距離感でサポートいただければ幸いです。

ご不明な点がございましたら、お気軽に塾までご連絡ください。
